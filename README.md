# AOR 第 25 组程序使用说明

## 编程语言

Python 3.7.0

## 环境配置

conda 4.9.2

## 第三方库版本

scipy 1.1.0

## 运行方式

配置好 Python3 环境后，将 main.py 和 Class.py 放于同一目录下，运行 main.py 即可。
